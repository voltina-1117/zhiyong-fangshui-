# 上线后：两人手机要做什么

> 链接部署好之后，**不用装 App**，按下面做一遍即可。  
> 开发前两人也可先申请 Server酱 SendKey，代码写好一起配进去。

---

## 一、分工：谁准备什么

| 事项 | 人 A（包工头） | 人 B（接单员） | 谁做 |
|------|----------------|----------------|------|
| Server酱 SendKey | 自己微信收推送 | 自己微信收推送 | **各做各的** |
| 管理账号登录 | 自己的用户名密码 | 自己的用户名密码 | 开发建好给你们 |
| 管理页快捷方式 | 加主屏幕/收藏 | 加主屏幕/收藏 | **各做各的** |
| 抖音客户链接 | 简介挂链接 | — | 通常一人管抖音 |

---

## 二、每人手机 5 步（约 10 分钟）

### 第 1 步：申请 Server酱（收新单微信响）

1. 手机浏览器打开 https://sct.ftqq.com  
2. 微信扫码登录  
3. 复制你的 **SendKey**（形如 `SCTxxxx`）  
4. 发给开发/负责人，配进系统（**不要发朋友圈**）

两人 **各申请各的**，系统会各推一条。

### 第 2 步：测一条推送

部署完成后，提交一条测试意向单 → 两人微信都应收到 Server酱 消息。

### 第 3 步：打开管理页并登录

1. 点 Server酱 里的链接，或打开：  
   `https://你的域名.vercel.app/admin.html`  
2. 输入你的 **用户名 + 密码**（开发会私下给你们）  
3. 登录成功后会记住约 30 天  

### 第 4 步：加到手机方便点开的地方

**推荐（iPhone）：**

- 用 Safari 打开 admin 链接 → 分享 → **添加到主屏幕** → 命名「防水接单」

**推荐（安卓）：**

- Chrome 打开 → 菜单 → **添加到主屏幕**

**备选：**

- 微信 → 文件传输助手 → 发给自己 admin 链接 → **收藏**

### 第 5 步：微信里别关 Server酱 通知

- 微信 → 设置 → 通知 → 确保允许通知  
- 把 Server酱 相关消息设为**星标/置顶**（可选）

---

## 三、客户链接放抖音（一人操作）

1. 复制留资页链接：  
   `https://你的域名.vercel.app/?from=douyin`  
2. 抖音 → 我 → 编辑主页 → **简介** → 粘贴链接  
3. 可选：用草料二维码等把链接做成图，放主页或视频  

**注意：** `admin.html` 链接 **不要** 放抖音公开位置，只给两人自己用。

---

## 四、日常怎么用

```
新单 → 两人微信同时响（Server酱）
     → 谁有空谁点链接
     → 拨号 → 改状态（系统记是谁操作的）
```

页面**不用一直挂后台**；漏单靠微信推送，不靠开着网页。

管理页**亮着时**会约 30 秒自动刷新，并有新单提示音（辅助）。

## 六、开发 / 部署顺序

```
1. ✅ 代码已在仓库（index / success / admin + Supabase 函数）
2. 注册 Supabase → SQL Editor 执行 supabase/deploy.sql（或 001 + 002 + 003 + seed_admins.sql）
3. 部署 Edge Functions create-lead、admin-api，配置环境变量：
   - SUPABASE_URL
   - SUPABASE_SERVICE_ROLE_KEY
   - SERVER_CHAN_KEY_1（人 A SendKey）
   - SERVER_CHAN_KEY_2（人 B SendKey）
   - ADMIN_BASE_URL（如 https://xxx.vercel.app/admin.html）
4. 复制 js/config.local.js.example → js/config.local.js，填入 Supabase 地址，设 DEMO_MODE: false
5. Vercel 导入本项目 → 部署 → 得到 HTTPS 链接
6. 两人按上文「手机 5 步」配置
7. 抖音简介挂：https://你的域名/?from=douyin
```
