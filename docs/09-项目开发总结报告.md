## 9.3 经验与教训

1. **需求明确是关键**：开发前界定「做什么」和「不做什么」，避免范围蔓延
2. **云开发降低部署门槛**：无需买服务器、配域名，适合个人快速上线
3. **先跑通核心流程**：表单能提交、数据能入库，再优化 UI
4. **AI 辅助编程提效**：自然语言描述需求 + 软件工程文档，可稳定生成可用代码
