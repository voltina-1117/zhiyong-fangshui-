# 用户操作手册

## 7.1 客户：从抖音怎么进来

| 方式 | 操作 |
|------|------|
| **点链接** | 抖音主页「简介」里的链接 → 自动跳转留资页 |
| **复制链接** | 简介/私信/评论区置顶 → 粘贴到浏览器或发给微信打开 |
| **扫二维码** | 视频/海报上的二维码 → 手机扫码打开（常用另一台手机扫） |

打开后：

1. **最上面**就是包工头电话，可直接拨打  
2. 看到小字：**上门费按距离征收**  
3. 填写：姓名、电话、位置、紧急程度、大致描述  
4. 勾选同意 → 提交 → 记意向单号，等平台联系  

---

## 7.2 平台（2 人管理）

### 账号

- 2 个独立账号（如：张师傅、接单小李），**权限一样**  
- 都能看单、改状态、录报价、确认收款  
- 系统会记**谁最后操作的**  

### 分工建议

| 角色 | 建议负责 |
|------|----------|
| 接单员 | 看 Server酱、首电、约查勘 |
| 包工头 | 上门查勘、报价、施工、确认收款 |

### 状态操作

同前：意向待联系 → … → 待平台付款 → 确认平台已收款 → 已完成
